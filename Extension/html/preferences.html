CYS Extension Preferences

        Select a visual theme to be applied to each page of the site (or stick with the default).         Select a visual theme to be applied to the storygame viewer.             The "standard" setting enables "My Stuff" customization, reply previewing, and other features that may make additional webpage requests. The "background" setting compounds this with toolbar notifications even when you are not browsing CYS (as long as you are still logged in). This is best left off if you have limited data or bandwidth.         When this box is checked, you will be alerted of your notifications via "badges" in the navigation bar rather than the default alert box.         When this box is checked, you will not recieve badge notifications for being tagged in forum posts.         Quality-of-life upgrades to the basic text editor when rich text is disabled. This includes syntax highlighting for both HTML and CYS Script, automatic bracket closing, dark and light themes, and more.         When this box is checked, the enhanced plain text editor will automatically suggest usernames following an @ for quicker tagging. Type at least three letters after to query a much longer list of members.         When this box is checked, the enhanced plain text editor automatically closes common brackets (' " ( [ {) as you type.         When this box is checked, the enhanced plain text editor automatically closes HTML tags as you type.         When this box is checked, the enhanced plain text editor realizes common programming character combinations as distinct, single forms (e.g. !=).         When this box is checked, HTML entered in the forum post editor is run through a normalization process before being submitted. This corrects any unclosed tags, escapes emoji and other complex characters, and may alter the quoting of attributes, but ultimately won't change the appearance of the rendered HTML. This feature applies to either plain text editor.         Check this box if you prefer the line breaks you type to be honored as such in posts, messages, etc.; leave unchecked to keep the default line-break-ignoring behavior. This feature works by automatically inserting additional HTML into the enhanced text editor. Note that disabling this feature will not remove the HTML in messages already posted.         Choose a language for your Notepad's syntax highlighting. Select the "plain text" option to disable highlighting.         By default, additional developer options are available when previewing your storygame from the editor. Checking this box enables these functionalities on all storygames.
